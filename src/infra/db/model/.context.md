# Database Model Context

## Purpose

Database models are GORM structs representing database tables. They are separate from domain entities and handle persistence concerns.

## Critical Rules

- **GORM tags**: All fields must have appropriate GORM tags
- **Primitive types**: Models use Go primitives and pointers, not value objects
- **ToEntity() method**: All models MUST provide transformation to domain entities
- **Constructor**: Provide `New{Model}` constructor for creating models
- **Table name**: Override `TableName()` for explicit table naming

## Model Structure

```go
type ModelName struct {
    ID            uint64    `gorm:"primaryKey"`
    RequiredField string    `gorm:"not null"`
    OptionalField *string
    CreatedAt     time.Time `gorm:"not null"`
}

func (ModelName) TableName() string {
    return "table_name"
}
```

## Constructor Pattern

```go
func NewModelName(
    id uint64,
    requiredField string,
    optionalField *string,
) ModelName {
    model := ModelName{
        RequiredField: requiredField,
        OptionalField: optionalField,
    }

    if id != 0 {
        model.ID = id
    }

    return model
}
```

## ToEntity() Transformation

All models MUST provide `ToEntity()` to convert to domain entities:

```go
func (model ModelName) ToEntity() (entity tkEntity.ModelName, err error) {
    // 1. Convert primitives to value objects with validation
    id, err := tkValueObject.NewModelId(model.ID)
    if err != nil {
        return entity, err
    }

    // 2. Handle optional fields
    var optionalVoPtr *tkValueObject.OptionalType
    if model.OptionalField != nil {
        vo, err := tkValueObject.NewOptionalType(*model.OptionalField)
        if err != nil {
            return entity, err
        }
        optionalVoPtr = &vo
    }

    // 3. Construct and return entity
    return tkEntity.NewModelName(id, requiredVo, optionalVoPtr, createdAt), nil
}
```

## Field Types

- **IDs**: `uint64` (converted to value object IDs in entities)
- **Strings**: `string` or `*string` for optional
- **Numbers**: `int`, `uint`, `float64`, etc.
- **Times**: `time.Time` (converted to `UnixTime` value object)
- **Slices**: `[]type` such as `[]string` or `[]uint64` with `gorm:"serializer:json"` tags
- **Relations**: Use GORM relationship tags

## Relationships

```go
type Parent struct {
    ID       uint64
    Children []Child // One-to-many
}

type Child struct {
    ID       uint64
    ParentID uint64
    Parent   Parent // Belongs-to
}
```
