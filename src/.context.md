Root source directory for the `tk` toolkit library — a reusable Go module (`github.com/goinfinite/tk`) providing domain primitives, infrastructure utilities, and presentation helpers. Organized as a three-layer architecture: domain, infra, and presentation.

## Summary

- domain/ — business logic layer: value objects, entities, DTOs, repository interfaces, and use cases
- infra/ — infrastructure layer: repository implementations, database services, and OS/network utilities
- presentation/ — presentation layer: HTTP request/response helpers, input parsing, and middleware

## Constraints

- MUST follow dependency direction: presentation -> infra -> domain. Domain MUST NOT depend on infra or presentation.
