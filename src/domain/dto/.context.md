<context path="src/domain/dto" updated="2026-07-10">

Data transfer objects for cross-layer communication. Pure data containers with no business logic. Fields use domain value objects where possible.

## Summary

- pagination.go — generic pagination parameters (page number, items per page, sort, last-seen-id)
- createActivityRecord.go — input DTO for creating an activity record
- deleteActivityRecord.go — input DTO for deleting activity records (supports filters)
- readActivityRecords.go — request and response DTOs for querying activity records with filters and pagination
- createHoneypotHit.go — input DTO for recording a honeypot hit (requester IP, path, class, count)
- readHoneypotBanDecision.go — request/response DTOs for checking if an IP is banned
- readHoneypotStatsReport.go — request/response DTOs for honeypot stats aggregation
- runHoneypotMaintenance.go — input DTO for maintenance operations (max entries, ban duration, stats interval)
- honeypotSettings.go — configuration DTO aggregating all honeypot VOs (no service dependencies)
