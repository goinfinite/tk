<context path="src/domain" updated="2026-03-09">

Business logic layer containing all domain primitives and rules. Completely independent of infrastructure and presentation concerns.

## Summary

- valueObject/ — immutable typed values with validation (e.g., IpAddress, Fqdn, UnixTime)
- entity/ — persistent business objects with identity (e.g., ActivityRecord, X509Certificate)
- dto/ — data transfer objects for cross-layer communication
- repository/ — interface definitions for data access (implementations live in infra/)
- useCase/ — business operations orchestrating entities and repositories

## Constraints

- MUST NOT import from `infra/` or `presentation/`.
- MUST use value objects instead of primitives for domain concepts.
