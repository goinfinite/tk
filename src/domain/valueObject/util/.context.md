Helper functions used by value object constructors for type conversion, regex extraction, and string normalization. Package name: `tkVoUtil`.

## Summary

- interfaceTo.go — converts `any` values to specific Go types (string, int, uint, float, bool, slices, etc.)
- interfaceTo_test.go — tests for type conversion functions
- namedGroupsExtractor.go — extracts named capture groups from regex matches into a map
- namedGroupsExtractor_test.go — tests for named groups extraction
- stringNormalizer.go — normalizes strings (trim, lowercase, remove extra whitespace, etc.)
- stringNormalizer_test.go — tests for string normalization
